#Gerador de CPF's

Este codigo tem como objetivo gerar CPF's para popular banco de dados de forma genérica.

## Uso
Dependências: <br>
```
$ g++
```
<br>

Para compilar use: <br>
```
$ g++ -o nomedesaida main.cpp
```
<br>

Para rodar use: <br>
```
$ ./nomedesaida <Numero_de_CPF's>
```
